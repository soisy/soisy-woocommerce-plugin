Con Soisy permetti ai tuoi clienti di pagare i loro ordini in comode rate.

== Description ==

Aumenta le conversioni offrendo il pagamento rateale Soisy: semplice, veloce, 100% online.
Un pagamento "più piccolo" stimola l'acquisto, favorisce le conversioni, aumenta l'importo medio dello scontrino.

**Come funziona ?**

  - Il cliente sceglie ciò che desidera e va al checkout.
  - Seleziona il metodo di pagamento Soisy, dopo aver visto simulazioni di pagamento rateale
  - Inizia la richiesta per pagarea a rate con Soisy


Per maggiori info e vantaggi per gli e-commerce [visita il nostro sito](https://www.soisy.it/ecommerce/).


== Screenshots ==

1. Il plugin installa anche il nostro widget di simulazione rateale che appare nelle pagine dei prodotti.
2. Il cliente può selezionare Soisy fra i metodi di pagamento
3. Il cliente vede una simulazione rateale nel proprio carrello
